# Ardi Nusawan Resume

## Preparation
1. Install LaTeX packages:
   ```bash
   sudo tlmgr update --self
   sudo tlmgr install titlesec marvosym enumitem fancyhdr
   ```
   Note: The `fullpage` package is obsolete and has been removed from the template.

## Build
```bash
pdflatex resume.tex
```
